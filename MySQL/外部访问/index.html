更改配置文件

访问 MySQL 的配置文件，如 linux 在/etc/mysql/mysql.conf.d/mysqld.cnf

找到 bind-address = 127.0.0.1

将其前面加上#，注释掉

按下Esc，输入:wq保存并退出
