Tkinter 文件操作

发表于 2024-03-10 更新于 2024-03-11 分类于 Python ， Tkinter

使用 filedialog 模块可以在图形界面中进行文件操作

filedialog 需要先创建一个窗口对象，然后调用相应的方法、

参数说明：

  • title：窗口标题
  • filetypes：文件类型（元组中的每个元素是一个文件类型，每个文件类型是一个元组，第一个元素是文件类型的描述，第二个元素是文件类型的扩展名，多个文件类型之间用分号隔开）
  • initialdir：初始目录
