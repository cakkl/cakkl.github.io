C++ 树

发表于 2024-03-06 分类于 C++ ， 数据结构 ， 树

树是一种非线性的数据结构，它是由 n（n>=1）个有限节点组成一个具有层次关系的集合。

结点：树的每个元素称为结点。
根节点：树的顶端结点称为根节点。
父节点：若一个结点含有子节点，则这个结点称为其子节点的父节点。
叶节点：没有子节点的结点称为叶节点。
节点的度：一个结点含有的子树的个数称为该结点的度。
树的度：一棵树中，最大的节点度称为树的度。
树的深度(高度)：树中结点的最大层次称为树的深度或高度。
