列出镜像

各个选项说明：

  • REPOSITORY：表示镜像的仓库源

  • TAG：镜像的标签

  • IMAGE ID：镜像 ID

  • CREATED：镜像创建时间

  • SIZE：镜像大小

查找镜像

  • NAME：镜像仓库源的名称

  • DESCRIPTION：镜像的描述

  • OFFICIAL：是否 docker 官方发布

  • stars：类似 Github 里面的 star，表示点赞、喜欢的意思。
